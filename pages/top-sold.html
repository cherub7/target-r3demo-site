How this demo works

Profiles — Michael (US), Neeraj (India), Chris (UK). Switching profile changes the active region.

Catalog — Clicking Buy sends a purchase event on item-purchase-mbox with orderId, orderTotal, productPurchasedId, and recsRegion for the active profile.

Recommendations — Calls Target mbox . When region personalisation is on, recsRegion is also sent with the recs request to surface region-specific results.

Target setup — Profile script recsAttributeRegion: return mbox.param("recsRegion");. Use ?recsDebug=1 to log raw offers to the console.

Settings

When enabled, sends recsRegion with the recommendation request and switches to the regional mbox r3demo-top-sold-by-region-recs.
