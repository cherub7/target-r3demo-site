How this demo works

Profiles — Michael (US), Neeraj (India), Chris (UK). Switching profile changes the active region.

Catalog — Each title click sends a view event on item-view-mbox with entity.* and recsRegion for the active profile.

Recommendations — Calls Target mbox . When region personalisation is on, recsRegion is sent to surface region-specific results.

Target setup — Profile script recsAttributeRegion: return mbox.param("recsRegion");. Add ?recsDebug=1 to log raw offers.

Settings

When enabled, sends recsRegion with the recommendation request. The section title and results update to reflect the active profile’s region.
